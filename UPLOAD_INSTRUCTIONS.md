# Upload Instructions

1. Extract this ZIP.
2. Open the folder inside it.
3. Upload all files and folders inside that folder to the root of your GitHub repository.
4. Commit changes to `main`.
5. In Netlify, use:

- Publish directory: `.`
- Functions directory: `netlify/functions`
- Build command: `echo No build needed`

6. Trigger deploy.
7. After deploy, test:
   - Edit a task subtask and confirm checked subtasks remain checked.
   - Edit a routine step and confirm today's checked steps remain checked.
   - Go to Customize and confirm Themed Days can be edited.
