# My Life Quest — Editable Recurring Routines v6

This version keeps the working app-style layout from v4/v5 and improves recurring routine editing.

## Added in v6

- Safer task subtask editing
- Safer recurring routine step editing
- Existing checkmarks are preserved when possible
- Renaming a step in place keeps its completion state
- Routine daily logs are cleaned up when a step is deleted
- Quick **Edit Subtasks** button for tasks
- Quick **Edit Steps** button for routines
- Themed Days can be edited from both **Routines** and **Customize**
- Routine edit screen includes an "Apply routine template changes" selector for future routine manager expansion

## Still included

- App-style pages/tabs
- Recurring routines
- Grace & Order themed days
- Today dashboard
- Smart Pomodoro task picker
- Pomodoro completion check-in
- Checkable subtasks
- Partial credit
- AI Plan My Day
- AI What Should I Do Next
- Custom dashboard order
- Custom page/list order
- Collapsible tasks, routines, projects, and goals
- Export/import backup
